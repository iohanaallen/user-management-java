Java User Management System (CRUD)

Sistema simples de gerenciamento de usuários (CRUD) desenvolvido em Java.

## Funcionalidades
- Cadastrar usuário
- Listar usuários
- Atualizar usuário
- Remover usuário

## Tecnologias
- Java
- Programação orientada a objetos

## Persistência
Os dados são salvos em arquivo local (`usuarios.txt`).

## Estrutura

src/
 ├── app/
 ├── model/
 └── service/

 ## Como executar
1. Abra o projeto no Eclipse
2. Execute a classe `Main.java`
3. Utilize o menu no console

## Backend

Este projeto consome a API disponível em:
